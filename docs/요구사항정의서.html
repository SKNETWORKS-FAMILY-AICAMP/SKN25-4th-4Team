🗄️ DB 스키마 (사용자 인증 + 대화 기록)
User (사용자)
컬럼타입설명
PKidINT사용자 ID
usernameVARCHAR사용자명 (unique)
emailVARCHAR이메일
passwordVARCHAR해시 비밀번호
created_atDATETIME가입일
ChatSession (대화 세션)
컬럼타입설명
PKidINT세션 ID
FKuser_idINT사용자 참조
titleVARCHAR첫 질문 (15자)
created_atDATETIME생성일
Message (대화 메시지)
컬럼타입설명
PKidINT메시지 ID
FKsession_idINT세션 참조
roleVARCHARuser / assistant
contentTEXT메시지 내용
has_paper_evidenceBOOL논문 근거 여부
weak_evidenceBOOL간접 근거 여부
paper_scoreFLOAT논문 관련도 점수
created_atDATETIME생성일
